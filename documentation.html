Introduction

The Quokka API에 오신 것을 환영합니다! 이 API를 사용하면 여러분의 웹사이트나 앱에 "세상에서 가장 행복한 동물" 쿼카의 사진을 쉽게 추가할 수 있습니다.

우리의 API는 RESTful 구조를 따르며, JSON 형식으로 데이터를 반환합니다. 지금 바로 무료로 시작해보세요!

Authentication

일부 엔드포인트는 API Key가 필요합니다. 하지만 GET /images/search와 같은 기본 기능은 인증 없이도 바로 사용할 수 있습니다.

x-api-key: YOUR_API_KEY_HERE

Search Images

랜덤한 쿼카 이미지를 가져오는 가장 기본적인 엔드포인트입니다.

GET https://api.thequokka.com/v1/images/search

Parameters

Parameter Type Description
limit Number 반환할 이미지 개수 (기본값: 1)
size String 이미지 크기 (small, med, full)

Example Response

[ { "id": "q_001", "url": "https://cdn.thequokka.com/images/q_001.jpg", "width": 800, "height": 600 } ]
